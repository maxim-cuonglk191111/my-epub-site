Thêm các secrets sau vào Worker để thấy trạng thái deploy thực tế trong tab Triển khai:

CF_API_TOKEN — CF API token (scope: Pages:Read)
CF_ACCOUNT_ID — Account ID từ CF Dashboard
CF_PAGES_PROJECT — Tên CF Pages project

Thư mục output/ không được commit vào GitHub — nó được tạo ra bởi CF Pages trong quá trình build và serve trực tiếp. Bạn sẽ không thấy nó trong repo.
